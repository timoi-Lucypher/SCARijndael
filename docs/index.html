Encryption of multiple plaintexts

Parameters

pts – (N, 4, 4) plaintext array

Returns

(N, 4, 4) ciphertexts

  • Example:

    key = np.array([
    [0x2b, 0x7e, 0x15, 0x16],
    [0x28, 0xae, 0xd2, 0xa6],
    [0xab, 0xf7, 0x15, 0x88],
    [0x09, 0xcf, 0x4f, 0x3c]
    ], dtype=np.uint8)

cipher = AES(key)
ciphertexts = np.random.randint(0, 256,
        size=(10, 4, 4), dtype=np.uint8)
plaintexts = cipher.decrypt(ciphertexts)
